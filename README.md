# 🤖 KnowledgeBot

AI-ассистент для Telegram, который отвечает на вопросы по вашим документам. Загружаете PDF или TXT → бот создаёт базу знаний → задаёте вопросы → получаете ответы с источниками.

## 🎯 Возможности

- 📚 **Загрузка документов** — PDF и TXT (до 20MB)
- 🔍 **RAG** — поиск релевантных фрагментов через ChromaDB
- 💬 **Умные ответы** — DeepSeek API с контекстом из документов
- 📖 **Источники** — каждый ответ ссылается на фрагменты документа
- 📋 **Управление базой** — список документов, очистка
- 🎨 **Клавиатура** — удобное меню в Telegram

## 📦 Установка

### 1. Клонировать репозиторий

```bash
git clone https://github.com/USERNAME/knowledgebot.git
cd knowledgebot
```

### 2. Создать виртуальное окружение

```bash
python3 -m venv venv
source venv/bin/activate  # macOS/Linux
# venv\Scripts\activate   # Windows
```

### 3. Установить зависимости

```bash
pip install -r requirements.txt
```

### 4. Настроить переменные окружения

```bash
cp .env.example .env
```

Отредактируйте `.env`:

```env
DEEPSEEK_API_KEY=your_key_here
TELEGRAM_BOT_TOKEN=your_bot_token_here
```

### 5. Получить API ключи

**DeepSeek API:**
1. Зайдите на https://platform.deepseek.com
2. Зарегистрируйтесь (5M бесплатных токенов)
3. Создайте API ключ
4. Скопируйте в `.env`

**Telegram Bot:**
1. Напишите @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен в `.env`

### 6. Запустить

```bash
python bot.py
```

## 📖 Как пользоваться

1. **Начало** — отправьте `/start`
2. **Загрузка** — нажмите «📚 Загрузить документ» и отправьте PDF/TXT
3. **Вопрос** — нажмите «❓ Задать вопрос» и напишите вопрос
4. **Бот ответит** на основе загруженных документов

### Примеры вопросов

- "Какие услуги вы предоставляете?"
- "Сколько стоит установка?"
- "Как связаться с поддержкой?"

## 🔧 Конфигурация

### Переменные окружения

| Переменная | Описание | По умолчанию |
|------------|----------|--------------|
| `DEEPSEEK_API_KEY` | API ключ DeepSeek | — |
| `TELEGRAM_BOT_TOKEN` | Токен Telegram-бота | — |
| `LLM_MODEL` | Модель DeepSeek | `deepseek-chat` |
| `CHROMA_DB_PATH` | Путь к ChromaDB | `./chroma_db` |
| `SYSTEM_PROMPT` | Системный промпт | Текущий |

### Системный промпт

Измените `SYSTEM_PROMPT` в `.env` для кастомизации поведения бота:

```env
SYSTEM_PROMPT=Ты юрист. Отвечай на основе загруженных документов. Цитируй статьи.
```
